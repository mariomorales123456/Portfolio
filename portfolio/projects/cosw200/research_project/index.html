AngularJS Research Project

Purpose

W3 Schools Angular JS

Angular JS unlike vanilla JavaScript seems to be far more intigrated with HTML. JavaScript would generally identify an html elemement(s) by ID or class, then add an event listener and then run a script based on that event. Angular JS let's you add variables, objects, arrays and scripts direclty to HTML.

I can see Angular JS being effective if you have script that can be intigrated locally in to HTML. It is also easier to increase interactivity with website users because you can have the script change attributes easily.

Tutorials

W3 Schools Learn Angular

The basics of the tutorials on Angular JS go over expressions which are a type of variable that can be plugged in inside HTML tags or inbewteen. Modules and directives define the script used for expressions. Data binding is a type of expression that specifically binds to an element as opposed to using the {{}} syntax.

Advanced concepts in the tutorial include controllers, filters and scopes. These act as the functions imported in to the document, and affect expressions based on where they are plugged in. It also has an HTTP service to interact with AJAX.
